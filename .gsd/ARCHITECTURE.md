## Overview

DropZone is a modern file-sharing web application. It allows users to securely upload files (single or batch), optionally password-protect them, and generate shareable links or QR codes. Files are encrypted on disk and automatically cleaned up after 24 hours. Emails containing share links can be sent directly from the platform.

## Data Flow

1. User drops files -> `script.js` prepares `FormData` (including optional password).
2. Frontend POSTs to `/api/upload`.
3. `server.js` uses `multer` to save a temp file to disk, generates a `shareId`, and encrypts the temp file to an `.enc` file utilizing an AES-256-CBC cipher with an IV.
4. `server.js` saves metadata to MongoDB and deletes the temp file.
5. User accesses download link (via URL or QR Code).
6. Request to `/download/:shareId` checks if file is active. Prompts for password if required.
7. Upon successful authentication, file is read, decrypted via stream, and piped to `res`.
